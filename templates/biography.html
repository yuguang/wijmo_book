Author Biography

Yuguang Zhang has worked as a web developer in a number of small companies and holds an undergraduate degree in Computer Science from the University of Waterloo. Notable companies where he has worked at include SociaLabra, a social media company, and SmartPager, a startup in mobile paging. He has expertise in developing interactive client-side applications with Knockout. As a personal project, he built the first IDE that runs Python in the browser, Python Fiddle, using Knockout and jQuery UI. In addition, he designed and programmed Fiddle Salad, a multi-language playground for rapid frontend development built with CoffeeScript and Django.

He is currently working as a research engineer at the University of Waterloo on a project called DataMill. It is a community-based easy to use service oriented benchmarking infrastructure for performance evaluation. As the main system administrator and developer of the team he installs Gentoo Linux on computers, develops distributed software to run on master and worker nodes, and programs interactive user interfaces. Wijmo's slider integration with Knockout has helped tremendously in developing the experiment submission form, where range sliders are used to select lower and upper bounds for factors such as page size, clock speed, and core count. As the web developer on the team, he programs in Python and CoffeeScript using the Django framework.
